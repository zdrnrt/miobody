# Проект Mio Body

## Стек
- Typescript
- Vite
- React, React Router, Tailwind

## Действия перед запуском проекта

Установка npm зависимостей

` npm install `

Запуск docker контейнера с бэкендом по адресу http://localhost:8000/api

Адрес к бэкенда можно отредактировать в файле .env

## Запуск проекта для разработки

` npm run dev `

Проект откроется по адресу http://localhost:3000/

## Запуск проекта для просмотра

` npm run build `

` npm run preview `

Проект откроется по адресу http://localhost:3000/
